🏂Markov Process

⚡ 问题背景

开始观看

在离散时间的 Markov 过程（Discrete-Time Markov Chains, DTMC）中，我们可以通过幂运算计算转移矩阵 PnP^nPn 来得到状态之间的转移概率。然而，在连续时间 Markov 过程（Continuous-Time Markov Chains, CTMC）中，这种方法不再适用。

因为在连续时间下，我们关注状态之间在非常短的时间 ttt 内的转移概率，通常记为 pjk(t)p_{jk}(t)pjk​(t)。这里的 pjk(t)p_{jk}(t)pjk​(t) 表示从状态 jjj 转移到状态 kkk 的概率。

如果状态空间 SSS 是无限的，分析会变得非常复杂。因此，我们常假设状态空间 ∣S∣|S|∣S∣ 是有限的，即 ∣S∣<∞|S| < \infty∣S∣<∞。

⛺定理1：转移率（Transition Rate）

定义：对于 j≠k∈Sj \neq k \in Sj=k∈S，有

qjk=qjrjk(1)q_{jk} = q_j r_{jk}\tag{1} qjk​=qj​rjk​(1)

其中：

  • qjkq_{jk}qjk​ 表示从状态 jjj 到状态 kkk 的转移率（transition rate）。
  • qjq_jqj​ 是从状态 jjj 出发的总体转移率（总转移到其他状态的速率）。
  • rjkr_{jk}rjk​ 是条件概率，即假设转移发生的情况下，从 jjj 转移到 kkk 的概率。

📝举例解释

假设你在超市里购物，每分钟会随机决定是否从一个货架走到另一个货架（这就是转移率 qjq_jqj​）。如果你选择转移，你可能会按某种概率走向特定的货架（对应 rjkr_{jk}rjk​）。
比如：

  • 你每分钟离开水果区的概率是 qj=0.5q_j = 0.5qj​=0.5。
  • 如果你离开水果区，你有 80% 的概率去零食区，20% 的概率去饮料区。
    因此：
  • 从水果区到零食区的转移率是 qjk=0.5×0.8=0.4q_{jk} = 0.5 \times 0.8 = 0.4qjk​=0.5×0.8=0.4。
  • 从水果区到饮料区的转移率是 qjk=0.5×0.2=0.1q_{jk} = 0.5 \times 0.2 = 0.1qjk​=0.5×0.2=0.1。

接下来在讲解新定理之前我们来了解以下几个知识点：

  1. Notation o(h)o(h)o(h) 的定义

如果满足以下条件：

g(h)h→0当 h→0。\frac{g(h)}{h} \to 0 \quad \text{当 } h \to 0。 hg(h)​→0当 h→0。

则函数 g(h)g(h)g(h) 是 o(h)o(h)o(h)。这意味着，当 hhh 变得非常小时，g(h)g(h)g(h) 的增长速度比 hhh 快速趋于 0。

🌰例子：

  • h2+3h3h^2 + 3h^3h2+3h3 是 o(h)o(h)o(h)，因为 h2+3h3h=h+3h2\frac{h^2 + 3h^3}{h} = h + 3h^2hh2+3h3​=h+3h2，当 h→0h \to 0h→0 时，趋于 0。
  • 2h+h22h + h^22h+h2 不是 o(h)o(h)o(h)，因为 2h+h2h=2+h\frac{2h + h^2}{h} = 2 + hh2h+h2​=2+h，当 h→0h \to 0h→0 时，不趋于 0。
  1. e−ahe^{-ah}e−ah 的近似

通过泰勒展开，我们有：

e−ah=1−ah+o(h)e^{-ah} = 1 - ah + o(h) e−ah=1−ah+o(h)

这表示在 hhh 很小时，指数函数可以用 1−ah1 - ah1−ah 来近似。

🌰例子：
想象你有一个非常小的时间间隔 hhh，假设你在 hhh 时间内完成某事件的概率可以通过指数分布建模。通过这个近似，可以快速估计转移概率。

  1. 概率 P(T≤h)P(T \leq h)P(T≤h)

如果 T∼exp(α)T \sim \text{exp}(\alpha)T∼exp(α)，即 TTT 遵循参数为 α\alphaα 的指数分布，则有：

P(T≤h)=1−e−αh=αh+o(h)P(T \leq h) = 1 - e^{-\alpha h} = \alpha h + o(h) P(T≤h)=1−e−αh=αh+o(h)

这表明，在非常短的时间间隔 hhh 内，事件发生的概率接近于 αh\alpha hαh。

🍻生活中的例子：
假设你观察公交车到站的时间间隔（符合指数分布），如果平均每分钟有 1 辆车到达（α=1\alpha = 1α=1），那么在非常短的时间 h=0.1h = 0.1h=0.1 分钟内车到达的概率大约是 αh=0.1\alpha h = 0.1αh=0.1（即 10%）。

⛺定理2

假设状态空间 ∣S∣=N<∞|S| = N < \infty∣S∣=N<∞，以下成立：
1.

pjj(h)=1−qjh+o(h)p_{jj}(h) = 1 - q_j h + o(h) pjj​(h)=1−qj​h+o(h)

pjk(h)=qjkh+o(h),∀j≠kp_{jk}(h) = q_{jk} h + o(h), \quad \forall j \neq k pjk​(h)=qjk​h+o(h),∀j=k

其中：

  • pjj(h)p_{jj}(h)pjj​(h) 是状态 jjj 在时间 hhh 内保持不变的概率。
  • pjk(h)p_{jk}(h)pjk​(h) 是在时间 hhh 内从状态 jjj 转移到状态 kkk 的概率。

我们称 qjkhq_{jk}hqjk​h 和 1−qjh1 - q_jh1−qj​h 为无限小转移概率（infinitesimal transition probabilities）。

证明部分

  1. 定义事件

    • A(h)A(h)A(h)：在时间间隔 (0,h](0, h](0,h] 内没有发生转移。
    • B(h)B(h)B(h)：在时间间隔 (0,h](0, h](0,h] 内发生至少两次转移。

  2. 计算 pjj(h)p_{jj}(h)pjj​(h)

    pjj(h)=P(X(h)=j,A(h)∣X(0)=j)+P(X(h)=j,B(h)∣X(0)=j)p_{jj}(h) = P(X(h) = j, A(h) \mid X(0) = j) + P(X(h) = j, B(h) \mid X(0) = j) pjj​(h)=P(X(h)=j,A(h)∣X(0)=j)+P(X(h)=j,B(h)∣X(0)=j)

    对于第一项，根据指数分布的性质，有：

    P(A(h)∣X(0)=j)=e−qjh=1−qjh+o(h)P(A(h) \mid X(0) = j) = e^{-q_j h} = 1 - q_j h + o(h) P(A(h)∣X(0)=j)=e−qj​h=1−qj​h+o(h)

    第二项是 P(B(h)∣X(0)=j)P(B(h) \mid X(0) = j)P(B(h)∣X(0)=j)，可以证明其为 o(h)o(h)o(h)，因为事件 B(h)B(h)B(h) 表示发生至少两次转移，其概率随着 hhh 减小时更快趋于 0。

    所以，结合两项：

    pjj(h)=1−qjh+o(h)p_{jj}(h) = 1 - q_j h + o(h) pjj​(h)=1−qj​h+o(h)

  3. 计算 pjk(h)p_{jk}(h)pjk​(h) （j≠kj \neq kj=k） 从状态 jjj 跳转到状态 kkk 的概率满足：

    pjk(h)=P(从 j 直接跳到 k 在时间 h 内)+P(B(h)∣X(0)=j)p_{jk}(h) = P(\text{从 $j$ 直接跳到 $k$ 在时间 $h$ 内}) + P(B(h) \mid X(0) = j) pjk​(h)=P(从 j 直接跳到 k 在时间 h 内)+P(B(h)∣X(0)=j)

    第一项是 qjkh+o(h)q_{jk} h + o(h)qjk​h+o(h)，第二项同样为 o(h)o(h)o(h)，因此：

    pjk(h)=qjkh+o(h)p_{jk}(h) = q_{jk} h + o(h) pjk​(h)=qjk​h+o(h)

生成矩阵 QQQ

定义生成矩阵 QQQ：

Q=(qjk),qjj=−qj,qjk=qjrjk,∀j≠kQ = (q_{jk}), \quad q_{jj} = -q_j, \quad q_{jk} = q_j r_{jk}, \quad \forall j \neq k Q=(qjk​),qjj​=−qj​,qjk​=qj​rjk​,∀j=k

生成矩阵的性质：

  • 每一行的元素和为 0：

    ∑k∈Sqjk=0\sum_{k \in S} q_{jk} = 0 k∈S∑​qjk​=0

    证明如下：

    ∑k∈Sqjk=qjj+∑k≠jqjk=−qj+qj∑k≠jrjk=−qj+qj=0\sum_{k \in S} q_{jk} = q_{jj} + \sum_{k \neq j} q_{jk} = -q_j + q_j \sum_{k \neq j} r_{jk} = -q_j + q_j = 0 k∈S∑​qjk​=qjj​+k=j∑​qjk​=−qj​+qj​k=j∑​rjk​=−qj​+qj​=0

  • qjj<0q_{jj} < 0qjj​<0，qjk≥0 (j≠k)q_{jk} \geq 0 \, (j \neq k)qjk​≥0(j=k)。

⛺定理3：Forward Equations

对于有限状态空间 ∣S∣<∞|S| < \infty∣S∣<∞，转移概率矩阵 P(t)P(t)P(t) 满足：

P′(t)=P(t)Q(3)P'(t) = P(t)Q \tag{3} P′(t)=P(t)Q(3)

其中：

  • P′(t)P'(t)P′(t) 表示转移矩阵 P(t)P(t)P(t) 中每个元素对时间 ttt 的导数。
  • QQQ 是生成矩阵。

这个公式被称为正向方程（Forward Equations）。

证明部分

  1. 目标
    我们需要证明每个元素 pjk(t)p_{jk}(t)pjk​(t) 满足：

    pjk′(t)=∑r∈Spjr(t)qrk.p'_{jk}(t) = \sum_{r \in S} p_{jr}(t)q_{rk}. pjk′​(t)=r∈S∑​pjr​(t)qrk​.

  2. 右导数的计算
    利用 2.5 中的结果，对任意 j,kj, kj,k：

    lim⁡h↓0pjk(t+h)−pjk(t)h\lim_{h \downarrow 0} \frac{p_{jk}(t+h) - p_{jk}(t)}{h} h↓0lim​hpjk​(t+h)−pjk​(t)​

    由转移概率的定义：

    pjk(t+h)=∑r∈Spjr(t)prk(h)p_{jk}(t+h) = \sum_{r \in S} p_{jr}(t) p_{rk}(h) pjk​(t+h)=r∈S∑​pjr​(t)prk​(h)

    因此：

    lim⁡h↓0pjk(t+h)−pjk(t)h=lim⁡h↓0∑r∈Spjr(t)prk(h)−pjk(t)h\lim_{h \downarrow 0} \frac{p_{jk}(t+h) - p_{jk}(t)}{h} = \lim_{h \downarrow 0} \frac{\sum_{r \in S} p_{jr}(t)p_{rk}(h) - p_{jk}(t)}{h} h↓0lim​hpjk​(t+h)−pjk​(t)​=h↓0lim​h∑r∈S​pjr​(t)prk​(h)−pjk​(t)​

  3. 分解计算
    将 r=kr = kr=k 和 r≠kr \neq kr=k 的部分拆分：

    =lim⁡h↓0∑r≠kpjr(t)prk(h)+pjk(t)pkk(h)−pjk(t)h.= \lim_{h \downarrow 0} \frac{\sum_{r \neq k} p_{jr}(t)p_{rk}(h) + p_{jk}(t)p_{kk}(h) - p_{jk}(t)}{h}. =h↓0lim​h∑r=k​pjr​(t)prk​(h)+pjk​(t)pkk​(h)−pjk​(t)​.

    使用 定理2 的结果：

    • 对于 r≠kr \neq kr=k，prk(h)=qrkh+o(h)p_{rk}(h) = q_{rk}h + o(h)prk​(h)=qrk​h+o(h)。
    • 对于 r=kr = kr=k，pkk(h)=1−qkh+o(h)p_{kk}(h) = 1 - q_k h + o(h)pkk​(h)=1−qk​h+o(h)。
      替换后得到：

      =lim⁡h↓0∑r≠kpjr(t)(qrkh+o(h))+pjk(t)(1−qkh+o(h))−pjk(t)h.= \lim_{h \downarrow 0} \frac{\sum_{r \neq k} p_{jr}(t)(q_{rk}h + o(h)) + p_{jk}(t)(1 - q_k h + o(h)) - p_{jk}(t)}{h}. =h↓0lim​h∑r=k​pjr​(t)(qrk​h+o(h))+pjk​(t)(1−qk​h+o(h))−pjk​(t)​.

  4. 化简
    将 hhh 提取出来：

    =lim⁡h↓0∑r≠kpjr(t)qrk+pjk(t)(−qk)+o(h).= \lim_{h \downarrow 0} \sum_{r \neq k} p_{jr}(t)q_{rk} + p_{jk}(t)(-q_k) + o(h). =h↓0lim​r=k∑​pjr​(t)qrk​+pjk​(t)(−qk​)+o(h).

    当 h→0h \to 0h→0 时，o(h)o(h)o(h) 消失，最终得到：

    pjk′(t)=∑r∈Spjr(t)qrk.p'_{jk}(t) = \sum_{r \in S} p_{jr}(t)q_{rk}. pjk′​(t)=r∈S∑​pjr​(t)qrk​.

  5. 左导数的计算
    同理，对于左导数：

    lim⁡h↓0pjk(t)−pjk(t−h)h\lim_{h \downarrow 0} \frac{p_{jk}(t) - p_{jk}(t-h)}{h} h↓0lim​hpjk​(t)−pjk​(t−h)​

    可使用类似的方法计算，最终也得到：

    pjk′(t)=∑r∈Spjr(t)qrk.p'_{jk}(t) = \sum_{r \in S} p_{jr}(t)q_{rk}. pjk′​(t)=r∈S∑​pjr​(t)qrk​.

  6. 导数的存在性
    由于左导数和右导数相等，因此 pjk(t)p_{jk}(t)pjk​(t) 的导数存在，并且满足：

    pjk′(t)=∑r∈Spjr(t)qrk.p'_{jk}(t) = \sum_{r \in S} p_{jr}(t)q_{rk}. pjk′​(t)=r∈S∑​pjr​(t)qrk​.

💡总结
通过以上证明，我们验证了转移概率矩阵的导数满足：

P′(t)=P(t)Q,P'(t) = P(t)Q, P′(t)=P(t)Q,

其中，生成矩阵 QQQ 表示每个状态之间的转移速率。

🪐Example: Flip-Flop Circuit

我们分析一个简单的两状态 Markov 过程，它的生成矩阵 QQQ 和转移概率矩阵 P(t)P(t)P(t) 满足 P′(t)=P(t)QP'(t) = P(t)QP′(t)=P(t)Q。

以下是他的推导过程：

🍦 定义生成矩阵 QQQ

根据题意：

  • q00=−q0=−λq_{00} = -q_0 = -\lambdaq00​=−q0​=−λ
  • q01=1⋅λ=λq_{01} = 1 \cdot \lambda = \lambdaq01​=1⋅λ=λ
  • q10=1⋅μ=μq_{10} = 1 \cdot \mu = \muq10​=1⋅μ=μ
  • q11=−q1=−μq_{11} = -q_1 = -\muq11​=−q1​=−μ

Q=(−λλμ−μ)Q = \begin{pmatrix} -\lambda & \lambda \\ \mu & -\mu \end{pmatrix} Q=(−λμ​λ−μ​)

🍦求解 P′(t)=P(t)QP'(t) = P(t)QP′(t)=P(t)Q

(p00′​(t)p10′​(t)​p01′​(t)p11′​(t)​)=(p00​(t)p10​(t)​p01​(t)p11​(t)​)(−λμ​λ−μ​).
