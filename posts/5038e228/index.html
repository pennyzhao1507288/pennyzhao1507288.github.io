🥕 Expectation

⛺什么是期望？

首先，我们假设有一个随机变量 XXX，它是在概率空间 (Ω,F,P)(\Omega, \mathcal{F}, \mathbb{P})(Ω,F,P) 上定义的。简单来说，这意味着 XXX 在某种“世界”中随机地取值，而我们希望了解它的“平均值”或“期望”，也就是 E(X)\mathbb{E}(X)E(X)。在这个世界中，Ω\OmegaΩ 是所有可能发生的情况的集合（称为样本空间），F\mathcal{F}F 是这些事件的集合，P\mathbb{P}P 是每个事件发生的概率。

我们可以将期望 E(X)\mathbb{E}(X)E(X) 想象成这样一个问题：假设你每次从一个袋子中随机取一个数 XXX，那么你取到的数的平均值会是多少呢？这就是我们要用 Lebesgue-Stieltjes 积分来计算的。

公式：

E(X):=∫ΩX dP\mathbb{E}(X) := \int_{\Omega} X \, d\mathbb{P} E(X):=∫Ω​XdP

这个积分表示了我们在所有可能的情况（样本空间 Ω\OmegaΩ）中，对 XXX 进行加权平均，而加权的方式由概率 P\mathbb{P}P 来决定。

Step 1: 简单随机变量的期望

假设 XXX 是一个简单的随机变量，这意味着它可以被分解成一些常数和指示函数的加和形式：

X=∑k=1∞ak1AkX = \sum_{k=1}^{\infty} a_k \mathbf{1}_{A_k} X=k=1∑∞​ak​1Ak​​

这里，aka_kak​ 是一些常数，1Ak\mathbf{1}_{A_k}1Ak​​ 是事件 AkA_kAk​ 是否发生的“指示函数”（它在 AkA_kAk​ 上为 1，其他地方为 0）。举个例子，如果 XXX 表示抛一枚硬币的结果，那么 XXX 可能取值为 0（表示反面）或 1（表示正面）。我们可以将 XXX 表示成 X=1⋅1{正面}+0⋅1{反面}X = 1 \cdot \mathbf{1}_{\{正面\}} + 0 \cdot \mathbf{1}_{\{反面\}}X=1⋅1{正面}​+0⋅1{反面}​。

对于这样的简单变量，期望可以定义为：

E(X)=∑k=1∞akP(Ak)\mathbb{E}(X) = \sum_{k=1}^{\infty} a_k \mathbb{P}(A_k) E(X)=k=1∑∞​ak​P(Ak​)

这相当于我们在每种可能的结果上取它的值 aka_kak​ 并乘以其发生的概率 P(Ak)\mathbb{P}(A_k)P(Ak​)。

Step 2: 一般非负随机变量的期望

如果 X≥0X \geq 0X≥0 是一个一般的随机变量，我们可以将它“分层”表示成一系列简单随机变量 XnX_nXn​ 的极限，其中：

Xn:=∑k=1∞k−12n1(k−12n<X≤k2n)X_n := \sum_{k=1}^{\infty} \frac{k-1}{2^n} \mathbf{1}\left(\frac{k-1}{2^n} < X \leq \frac{k}{2^n}\right) Xn​:=k=1∑∞​2nk−1​1(2nk−1​<X≤2nk​)

通过这种方式，我们逐渐“逼近” XXX，并定义 E(X)\mathbb{E}(X)E(X) 为 XnX_nXn​ 的期望的极限：

E(X)=lim⁡n→∞E(Xn)\mathbb{E}(X) = \lim_{n \to \infty} \mathbb{E}(X_n) E(X)=n→∞lim​E(Xn​)

可以想象我们在无限细分 XXX 的可能取值区间，使得 XnX_nXn​ 越来越接近真实的 XXX。比如，把一个随机数值的范围不断切分得更小，以便获得更精确的期望值。

Step 3: 一般随机变量的期望

对于任意随机变量 XXX，我们将其拆分为正的部分和负的部分：

X=X+−X−X = X^+ - X^- X=X+−X−

其中 X+:=X∨0X^+ := X \vee 0X+:=X∨0 表示 XXX 的非负部分，而 X−:=(−X)∨0X^- := (-X) \vee 0X−:=(−X)∨0 表示 XXX 的非正部分。

于是，我们可以定义 XXX 的期望为：

E(X):=∫ΩX dP:=∫ΩX+−∫ΩX−  dP:=E(X+)−E(X−)(1)\mathbb{E}(X) := \int_{\Omega} X \, d\mathbb{P}:= \int_{\Omega} X^+-\int_{\Omega} X^- \ \, d\mathbb{P}:=\mathbb{E}(X^+) - \mathbb{E}(X^-)\tag{1} E(X):=∫Ω​XdP:=∫Ω​X+−∫Ω​X− dP:=E(X+)−E(X−)(1)

这种方法的本质是，我们将一个有正负值的随机变量分解成纯粹的非负值的部分，从而使得每一部分的期望都可计算，然后通过减法得到 XXX 的期望。

可以把这种分解的过程类比成我们计算净资产：假设 XXX 是你的总资产值（有正有负），X+X^+X+ 是你所有的资产，而 X−X^-X− 是你的所有负债，那么 XXX 的期望就是你资产的期望减去负债的期望，得到净资产的期望值。

接下来讲一下可积随机变量，这个概念告诉我们，在什么情况下我们能够计算一个随机变量的期望，并且这个期望是有限的。

🥕 Integrable random variables

⛺可积随机变量的定义

假设我们有一个随机变量 XXX，如果 XXX 的期望 E(X)\mathbb{E}(X)E(X) 存在且是有限的，那么我们称 XXX 为一个可积随机变量。这也就是为什么在(14)中，我们有了一个条件来判断期望是否存在并且是有限的。

⛺公式分析

  1. 绝对值的分解：我们将 ∣X∣|X|∣X∣ 表示为 X+X^+X+ 和 X−X^-X− 的和：

    ∣X∣=X++X−(2)|X| = X^+ + X^- \tag{2} ∣X∣=X++X−(2)

    这意味着 XXX 的绝对值等于它的正部分和负部分之和。可以理解为，将 XXX 的正值部分和负值部分拆开，然后把这两部分加起来。

  2. 期望的绝对值：通过对 ∣X∣|X|∣X∣ 求期望，我们得到：

    E∣X∣=EX++EX−(3)\mathbb{E}|X| = \mathbb{E}X^+ + \mathbb{E}X^-\tag{3} E∣X∣=EX++EX−(3)

    这条公式表明，如果 XXX 的正部分和负部分的期望都是有限的，那么 XXX 的期望就是有限的。换句话说，如果 EX+\mathbb{E}X^+EX+ 和 EX−\mathbb{E}X^-EX− 都存在且是有限值，那么 E∣X∣\mathbb{E}|X|E∣X∣ 也是有限的。

  3. 判断期望存在的条件：从(1)和(3)，可以得到以下等价条件：

    E(X) 存在且有限  ⟺  E∣X∣<∞  ⟺  EX+<∞ 且 EX−<∞(4)\mathbb{E}(X) \text{ 存在且有限} \iff \mathbb{E}|X| < \infty \iff \mathbb{E}X^+ < \infty \, \text{且} \, \mathbb{E}X^- < \infty \tag{4} E(X) 存在且有限⟺E∣X∣<∞⟺EX+<∞且EX−<∞(4)

    这个条件的意思是，只要 XXX 的正部分和负部分的期望都存在且有限，那么整个 XXX 的期望就是存在且有限的。

🌟可以将这个情况想象成一个“资产负债表”的情景：XXX 是你的净资产，X+X^+X+ 是你所有的资产，X−X^-X− 是你的负债。如果你的资产和负债都不是无限大的（也就是 EX+<∞\mathbb{E}X^+ < \inftyEX+<∞ 且 EX−<∞\mathbb{E}X^- < \inftyEX−<∞），那么你的净资产 XXX 的期望值就是有限的。这样，我们就可以对你的资产状况做出有意义的统计分析了。

这段定理的核心是理解在什么情况下我们可以计算一个随机变量的期望值，并且保证这个值是有限的。只要正部分和负部分的期望都是有限的，那么这个随机变量就称为可积随机变量(integrable random variable)或者that its expectation/mean is ﬁnite，也就是我们可以合理地去计算它的期望。

🥕 Lebesgue-Stieltjes (LS) 积分

⛺LS积分的构造

首先，我们有一个测度空间 (S,B,μ)(S, \mathcal{B}, \mu)(S,B,μ)，其中：

  • SSS 是我们讨论的空间（可以是实数集 R\mathbb{R}R 或更广泛的集合）。
  • B\mathcal{B}B 是 SSS 上的一个 σ\sigmaσ-代数，也就是说它是 SSS 的所有可能“事件”或“集合”的一个集合。
  • μ\muμ 是定义在 B\mathcal{B}B 上的一个测度（它可以不是概率测度）。测度的作用是给集合赋予一个“大小”或“体积”。

假设我们有一个函数 f:S→Rf: S \rightarrow \mathbb{R}f:S→R，并且它在测度 μ\muμ 下是可测的。我们定义 fff 关于 μ\muμ 的 LS 积分为：

∫Sf dμ=∫Sf(s) dμ(s)\int_S f \, d\mu = \int_S f(s) \, d\mu(s) ∫S​fdμ=∫S​f(s)dμ(s)

这个积分的含义是我们对 SSS 空间中的每一个“点” sss，按照测度 μ\muμ 的权重来“加权”函数 fff 的值。这种加权积分方式比传统的 Riemann 积分更灵活，因为它允许处理的函数和集合更加复杂。

⛺特殊情况：Lebesgue 积分

当 S=RS = \mathbb{R}S=R，B=B(R)\mathcal{B} = \mathcal{B}(\mathbb{R})B=B(R) 表示实数上的 Borel 集合，μ=λ\mu = \lambdaμ=λ 为 Lebesgue 测度时，我们得到 Lebesgue 积分。这时，LS 积分就简化为普通的 Lebesgue 积分，记作：

∫Rf dλ=∫Rf(x) dx\int_{\mathbb{R}} f \, d\lambda = \int_{\mathbb{R}} f(x) \, dx ∫R​fdλ=∫R​f(x)dx

Lebesgue 积分的出现是为了扩展 Riemann 积分，使得我们可以对更加复杂的函数进行积分，比如一些在传统 Riemann 积分下无法处理的函数。

⛺特殊情况：Lebesgue-Stieltjes 积分

当测度 μ\muμ 是 λF\lambda_FλF​，其中 λF\lambda_FλF​ 是一个由分布函数 FFF 生成的测度时，我们得到的是 Lebesgue-Stieltjes 积分：

∫Rf dλF=∫Rf(x) dF(x)\int_{\mathbb{R}} f \, d\lambda_{F} = \int_{\mathbb{R}} f(x) \, dF(x) ∫R​fdλF​=∫R​f(x)dF(x)

分布函数 FFF 描述了随机变量的累积分布，而积分 ∫f dF\int f \, dF∫fdF 表示对 fff 进行加权平均，权重由 FFF 确定。

🥕 The standard machine

这个方法是用来证明一些“线性”陈述的，特别是关于期望的等式。

⛺标准机器的应用

标准机器的流程如下：假设我们想要证明某个关于随机变量 XXX 的函数 f(X)f(X)f(X) 的期望 E[f(X)]\mathbb{E}[f(X)]E[f(X)] 可以通过积分的形式表示为

E[f(X)]=∫Ωf(X) dP=∫Rf(x) dPX(x)=∫Rf(x) dFX(x)(5)\mathbb{E}[f(X)] = \int_{\Omega} f(X) \, d\mathbb{P} = \int_{\mathbb{R}} f(x) \, d\mathbb{P}_X(x) = \int_{\mathbb{R}} f(x) \, dF_X(x)\tag{5} E[f(X)]=∫Ω​f(X)dP=∫R​f(x)dPX​(x)=∫R​f(x)dFX​(x)(5)

其中：

  • PX\mathbb{P}_XPX​ 是随机变量 XXX 的分布，即 XXX 的“规律”或“法则”。
  • FXF_XFX​ 是 XXX 的累积分布函数，描述了 XXX 在不同区间上取值的概率累积。

这个等式意味着：对随机变量 XXX 的期望可以通过 XXX 的概率分布来计算。它告诉我们，计算期望时可以直接使用分布函数 FXF_XFX​ 来加权求和，而不必回到原始的概率空间 Ω\OmegaΩ。

使用标准机器的步骤

为了证明上面的等式（公式 (5)），我们会按照以下步骤来逐步处理不同的函数类型：

  1. 从简单函数开始：先考虑 fff 是某个 Borel 集合的指示函数。这样可以通过简单的事件计算来理解。
  2. 逐步扩展到更复杂的函数：然后我们将 fff 扩展到简单函数（可以表示为有限个常数和指示函数的和），之后是非负的函数。
  3. 最终处理一般实值函数：最后，处理任意的实值函数，这一步是通过限制条件和取极限完成的。

这样分步操作的优点是，先处理简单的情况，然后逐渐增加复杂性，直到覆盖一般的情况。这种方法可以确保证明过程清晰、严谨，并且可以用来处理非常一般的随机变量和函数。

“标准机器”方法是一种有条理的证明工具，特别适用于在测度论中处理关于期望的线性陈述。通过分步处理，我们可以从最简单的指示函数出发，最终证明出一般函数的期望公式。这种技巧在测度论和概率论中非常实用，可以用来证明各种关于期望和积分的等式。

🥕 Monotone Convergence Theorem

单调收敛定理（Monotone Convergence Theorem），是测度论中一个非常重要的工具，用于分析一列随机变量的期望收敛行为。

⛺定理内容

假设 (Xn)(X_n)(Xn​) 是一列随机变量，并且 XXX 是一个随机变量，定义在同一个概率空间 (Ω,F,P)(\Omega, \mathcal{F}, \mathbb{P})(Ω,F,P) 上。若满足以下两个条件：

  1. 当n→∞n \to \inftyn→∞，则Xn≥0X_n \geq 0Xn​≥0 且 Xn↑XX_n \uparrow XXn​↑X [6]，即 (Xn)(X_n)(Xn​) 是一个非减序列并且收敛到 XXX；
  2. 则我们有 E(Xn)↑E(X)\mathbb{E}(X_n) \uparrow \mathbb{E}(X)E(Xn​)↑E(X)，即 (Xn)(X_n)(Xn​) 的期望随着 nnn 增大单调收敛到 XXX 的期望。

形式化地表示为：

lim⁡n→∞E(Xn)=E(X)\lim_{n \to \infty} \mathbb{E}(X_n) = \mathbb{E}(X) n→∞lim​E(Xn​)=E(X)

证明过程

这个证明的核心思想是通过构造一列简单的函数 (Zn)(Z_n)(Zn​) 来逼近 XXX，并验证在这种构造下，ZnZ_nZn​ 和 XnX_nXn​ 的期望逐步收敛到 E(X)\mathbb{E}(X)E(X)。证明分为几个关键步骤。

第一步：定义序列 ZnZ_nZn​

定义 ZnZ_nZn​ 为：

Zn:=∑k=1∞k−12n1(k−12n<X≤k2n)Z_n := \sum_{k=1}^{\infty} \frac{k-1}{2^n} \mathbf{1}\left(\frac{k-1}{2^n} < X \leq \frac{k}{2^n}\right) Zn​:=k=1∑∞​2nk−1​1(2nk−1​<X≤2nk​)

其中 1\mathbf{1}1 表示指示函数，即对于满足 k−12n<X≤k2n\frac{k-1}{2^n} < X \leq \frac{k}{2^n}2nk−1​<X≤2nk​ 的情形，1\mathbf{1}1 取值为 1，其他情形取值为 0。这样构造的 ZnZ_nZn​ 是一个简单函数，它逼近 XXX。

第二步：验证 Zn↑XZ_n \uparrow XZn​↑X 且 E(Zn)↑E(X)\mathbb{E}(Z_n) \uparrow \mathbb{E}(X)E(Zn​)↑E(X)

由于 Zn↑XZ_n \uparrow XZn​↑X（因为 ZnZ_nZn​ 随着 nnn 增大不断逼近 XXX），根据单调性，我们有 E(Zn)↑E(X)\mathbb{E}(Z_n) \uparrow \mathbb{E}(X)E(Zn​)↑E(X)。所以可以找到一个足够大的 nnn 使得

L<E(Zn)L < \mathbb{E}(Z_n) L<E(Zn​)

其中 L≤E(X)L \leq \mathbb{E}(X)L≤E(X) 是一个假设值，目的是为了构建一个矛盾。

第三步：重写 ZnZ_nZn​

通过以下过程将 ZnZ_nZn​ 进一步展开为简单的项：

Zn=∑k=1∞k−12n1(k−12n<X)−∑k=1∞k−12n1(k2n<X)Z_n = \sum_{k=1}^{\infty} \frac{k-1}{2^n} \mathbf{1}\left(\frac{k-1}{2^n} < X\right) - \sum_{k=1}^{\infty} \frac{k-1}{2^n} \mathbf{1}\left(\frac{k}{2^n} < X\right) Zn​=k=1∑∞​2nk−1​1(2nk−1​<X)−k=1∑∞​2nk−1​1(2nk​<X)

接下来对 ZnZ_nZn​ 进行一系列分拆、移项，得到

Zn=∑k=1∞12n1(X>k2n)Z_n = \sum_{k=1}^{\infty} \frac{1}{2^n} \mathbf{1}\left(X > \frac{k}{2^n}\right) Zn​=k=1∑∞​2n1​1(X>2nk​)

第四步：利用公式[6]

通过以上展开，我们可以得到

E(Zn)=∑k=1∞12nP(X>k2n)\mathbb{E}(Z_n) = \sum_{k=1}^{\infty} \frac{1}{2^n} \mathbb{P}\left(X > \frac{k}{2^n}\right) E(Zn​)=k=1∑∞​2n1​P(X>2nk​)

所以，只要 Xn↑XX_n \uparrow XXn​↑X，我们就可以证明 E(Xn)↑E(X)\mathbb{E}(X_n) \uparrow \mathbb{E}(X)E(Xn​)↑E(X)，这完成了证明。

⛺Fatou's Lemma

引理内容

设 (Xn)(X_n)(Xn​) 是定义在同一概率空间 (Ω,F,P)(\Omega, \mathcal{F}, \mathbb{P})(Ω,F,P) 上的随机变量序列。如果满足以下条件，则有：

  1. 如果 Xn≥0X_n \geq 0Xn​≥0 对于所有 n≥1n \geq 1n≥1，则

    E(lim inf⁡n→∞Xn)≤lim inf⁡n→∞E(Xn)(7)\mathbb{E} \left( \liminf_{n \to \infty} X_n \right) \leq \liminf_{n \to \infty} \mathbb{E}(X_n)\tag{7} E(n→∞liminf​Xn​)≤n→∞liminf​E(Xn​)(7)

  2. 如果 Xn≤0X_n \leq 0Xn​≤0 对于所有 n≥1n \geq 1n≥1，则

    E(lim sup⁡n→∞Xn)≥lim sup⁡n→∞E(Xn)(8)\mathbb{E} \left( \limsup_{n \to \infty} X_n \right) \geq \limsup_{n \to \infty} \mathbb{E}(X_n)\tag{8} E(n→∞limsup​Xn​)≥n→∞limsup​E(Xn​)(8)

这些不等式的意义在于，它们描述了当我们对一个随机变量序列取下极限或上极限后，再计算期望时的行为。

对于(7)

E(lim inf⁡n→∞Xn)≤lim inf⁡n→∞E(Xn)\mathbb{E} \left( \liminf_{n \to \infty} X_n \right) \leq \liminf_{n \to \infty} \mathbb{E}(X_n) E(n→∞liminf​Xn​)≤n→∞liminf​E(Xn​)

  2. 因此，我们可以得到：

    E(lim inf⁡n→∞Xn)=E(lim⁡n→∞inf⁡k≥nXk)=lim⁡n→∞E(inf⁡k≥nXk)\mathbb{E} \left( \liminf_{n \to \infty} X_n \right) = \mathbb{E} \left( \lim_{n \to \infty} \inf_{k \geq n} X_k \right) = \lim_{n \to \infty} \mathbb{E} \left( \inf_{k \geq n} X_k \right) E(n→∞liminf​Xn​)=E(n→∞lim​k≥ninf​Xk​)=n→∞lim​E(k≥ninf​Xk​)

    这里使用了单调收敛定理，因为 inf⁡k≥nXk\inf_{k \geq n} X_kinfk≥n​Xk​ 是一个非减序列。

  3. 根据期望的性质，可以得到不等式：

    E(inf⁡k≥nXk)≤inf⁡k≥nE(Xk)\mathbb{E} \left( \inf_{k \geq n} X_k \right) \leq \inf_{k \geq n} \mathbb{E}(X_k) E(k≥ninf​Xk​)≤k≥ninf​E(Xk​)

  4. 于是，有：

🥕 Dominated Convergence Theorem

主导收敛定理（Dominated Convergence Theorem, DCT）通常用于处理随机变量序列的期望的极限交换问题。该定理可在一定条件下将极限操作和积分操作交换顺序。

引理内容

假设 (Xn)(X_n)(Xn​) 和 XXX 是定义在概率空间 (Ω,F,P)(\Omega, \mathcal{F}, \mathbb{P})(Ω,F,P) 上的随机变量序列。若满足以下条件：

  1. Xn→XX_n \to XXn​→X 几乎必然地收敛于 XXX，即

    Xn→X当n→∞X_n \to X \quad \text{当} \quad n \to \infty Xn​→X当n→∞

  2. 存在一个可积随机变量 ZZZ 使得对所有 n≥1n \geq 1n≥1，

    ∣Xn∣≤Z|X_n| \leq Z ∣Xn​∣≤Z

则我们可以得到以下结论：

lim⁡n→∞E(Xn)=E(X)\lim_{n \to \infty} \mathbb{E}(X_n) = \mathbb{E}(X) n→∞lim​E(Xn​)=E(X)

证明过程

为了证明这个定理，我们主要应用法图引理（Fatou’s Lemma）以及对期望的极限性质的分析。

  1. 利用有界性条件：由于我们知道 ∣Xn∣≤Z|X_n| \leq Z∣Xn​∣≤Z 且 ZZZ 是一个可积的随机变量，因此我们可以构造一个新的随机变量 Z~\tilde{Z}Z~ 使得 Z~=2Z\tilde{Z} = 2ZZ~=2Z。这样，我们有

    ∣Xn−X∣≤∣Xn∣+∣X∣≤2Z=Z~|X_n - X| \leq |X_n| + |X| \leq 2Z = \tilde{Z} ∣Xn​−X∣≤∣Xn​∣+∣X∣≤2Z=Z~

    这说明 (Xn−X)(X_n - X)(Xn​−X) 也被 Z~\tilde{Z}Z~ 支配。

  2. 应用Fatou引理：根据法图引理，我们可以得到

    E(lim inf⁡n→∞∣Xn−X∣)≤lim inf⁡n→∞E(∣Xn−X∣)\mathbb{E} \left( \liminf_{n \to \infty} |X_n - X| \right) \leq \liminf_{n \to \infty} \mathbb{E}(|X_n - X|) E(n→∞liminf​∣Xn​−X∣)≤n→∞liminf​E(∣Xn​−X∣)

    由于 Xn→XX_n \to XXn​→X，因此 lim inf⁡n→∞∣Xn−X∣=0\liminf_{n \to \infty} |X_n - X| = 0liminfn→∞​∣Xn​−X∣=0。于是有

    E(0)≤lim inf⁡n→∞E(∣Xn−X∣)\mathbb{E}(0) \leq \liminf_{n \to \infty} \mathbb{E}(|X_n - X|) E(0)≤n→∞liminf​E(∣Xn​−X∣)

    因此得到

    lim inf⁡n→∞E(∣Xn−X∣)=0\liminf_{n \to \infty} \mathbb{E}(|X_n - X|) = 0 n→∞liminf​E(∣Xn​−X∣)=0

  3. 证明lim sup⁡\limsuplimsup也为零：另一方面，我们也可以得到

    0≤lim sup⁡n→∞E(∣Xn−X∣)≤E(0)=00 \leq \limsup_{n \to \infty} \mathbb{E}(|X_n - X|) \leq \mathbb{E}(0) = 0 0≤n→∞limsup​E(∣Xn​−X∣)≤E(0)=0

    这意味着

    lim⁡n→∞E(∣Xn−X∣)=0\lim_{n \to \infty} \mathbb{E}(|X_n - X|) = 0 n→∞lim​E(∣Xn​−X∣)=0

  4. 推导期望的极限：由此我们可以得到

    lim⁡n→∞E(Xn)=E(X)\lim_{n \to \infty} \mathbb{E}(X_n) = \mathbb{E}(X) n→∞lim​E(Xn​)=E(X)

这完成了主导收敛定理的证明。

主导收敛定理，使我们能够在满足一定条件（有界性和收敛性）的情况下，将求期望和取极限这两个操作交换顺序。

在主导收敛定理中，∣Xn∣≤Z|X_n| \leq Z∣Xn​∣≤Z不可省略。如果我们想让 Xn→XX_n \to XXn​→X 几乎必然地收敛推出结论 E(Xn)→E(X)\mathbb{E}(X_n) \to \mathbb{E}(X)E(Xn​)→E(X)，则必须有 ∣Xn∣≤Z|X_n| \leq Z∣Xn​∣≤Z的存在，即存在一个可积随机变量 ZZZ 使得 ∣Xn∣≤Z|X_n| \leq Z∣Xn​∣≤Z 对所有 n≥1n \geq 1n≥1 成立。主导条件提供了一个“上界”，保证了收敛的随机变量序列 (Xn)(X_n)(Xn​) 在期望上也是受控的。

通过证明过程，我们实际上得到了一个更强的结论即：

lim⁡n→∞E(∣Xn−X∣)=0\lim_{n \to \infty} \mathbb{E}(|X_n - X|) = 0 n→∞lim​E(∣Xn​−X∣)=0

Scheffé引理在测度论中提供了一个重要的结果，说明了当一列随机变量的积分或期望收敛时，该随机变量的L1L^1L1收敛性（即期望的绝对差的收敛性）。以下是对引理内容及其证明的详细讲解。
